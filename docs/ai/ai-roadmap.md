# AI Roadmap

## Release Gates

Before any Phase 2 capability:

- Phase 1 data and calculations are stable.
- Workspace authorization and audit logging are complete.
- Tool schemas and domain services are tested.
- Privacy/provider decisions are approved.
- Evaluation datasets exclude uncontrolled sensitive production data.
- Safety, rollback, monitoring, and feature flags are ready.

## Stage 0: Phase 1 Presence

- Flow orb and destination
- Honest Coming Soon content
- Capability education
- Feedback and consent-based Notify Me

## Stage 1: Low-Risk Assistance

- Read-only natural-language search
- Monthly summaries
- Explanations of reports
- Merchant cleanup suggestions
- Categorization suggestions with confidence

No automatic writes.

## Stage 2: Detection and Recommendations

- Duplicate detection
- Receipt OCR preview
- Bill/subscription detection
- Unusual-spending alerts
- Budget and savings recommendations
- Cash-flow forecasting

Users correct suggestions; corrections inform evaluation without silently
becoming training consent.

## Stage 3: User-Approved Actions

- Propose transaction creation
- Propose budgets
- Propose savings/debt goals
- Propose reminders/categories

Every action uses the approval workflow and ordinary domain services.

## Stage 4: Household Assistance

- Shared-workspace summaries
- Collaborative budget/goal support
- Member-aware explanations
- Privacy-safe household recommendations

## Evaluation

Measure task success, factual grounding, calculation agreement, unauthorized
data access, unsafe advice, tool argument validity, confirmation integrity,
latency, and user correction rates.

## Deferred Decisions

- Model/provider and regional hosting
- Retrieval/vector storage need
- Conversation retention duration
- On-device guest AI feasibility
- Cost/usage limits and premium packaging
- Human escalation/support path
