NidhiFlow SVG Package

Files:
1. NidhiFlow_Home_v1.svg
   Full frozen Home screen design.

2. NidhiFlow_Components_v1.svg
   Reusable component sheet.

How to import into Figma:
1. Open Figma.
2. Drag NidhiFlow_Home_v1.svg onto the canvas.
3. Right-click the imported SVG and choose Ungroup.
4. Edit text, colors, icons, and shapes directly.
5. Drag NidhiFlow_Components_v1.svg onto another page.
6. Convert repeated groups into Figma components.

Note:
SVG is editable in Figma after ungrouping, but it will not automatically create Auto Layout, variants, or design tokens.
