# NidhiFlow Home Figma Package

This package contains:

- NidhiFlow_Home_Figma_Spec.json
- nidhiflow_leaf_logo.svg
- flow_orbit_button.svg
- goal_goa_trip.svg

How to use:

1. Open Figma.
2. Drag the SVG assets into your Figma canvas.
3. Use the JSON as the screen/component blueprint.
4. Rebuild using frames, auto-layout, and components based on the coordinates and tokens.
5. Convert repeated items into components: Card, Action Card, Flow Button, Nav Item, Stat Row, Transaction Row.

Note:
This is not a native .fig file. Figma does not support importing arbitrary JSON as a design file without a plugin/custom script.
